# Excel-Sales-Dashboard

This repository contains an interactive **Excel Sales Dashboard** that displays sales performance for different regions and executives.  

## Features
- Interactive slicers to filter data by region  
- Bar, Pie, and Line charts for visual representation  
- Target achievement and total sales metrics  
- Macro-enabled file for automation

## How to Use
1. Open the Excel file in **Microsoft Excel** (macro-enabled).  
2. Use the slicers to filter sales data by region.  
3. Explore charts to analyze sales performance.  

## Notes
- This project is for learning and demonstration purposes.  
- Ensure macros are enabled to use interactive features.
